# Infrastructure Components

The infrastructure layer provides the foundational services that power the entire platform.

## Core Components

- **[Cilium](cilium/index.md)**: eBPF-based CNI with Gateway API support and L7 proxy capabilities
- **[Cert Manager](cert-manager/index.md)**: Cloud-native certificate management for Kubernetes
- **[Vault](vault/index.md)**: Secrets management and data protection platform
- **[External Secrets](external-secrets/index.md)**: Synchronize secrets from external sources into Kubernetes
- **[ArgoCD](argocd/index.md)**: Declarative GitOps continuous delivery for Kubernetes

## Component Details

Each component is thoroughly documented with:
- Version information
- Configuration parameters
- Resource requirements
- Best practices
