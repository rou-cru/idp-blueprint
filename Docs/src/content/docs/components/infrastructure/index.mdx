The infrastructure layer provides the foundational services that power the entire platform.

## Core Components

- **[Cilium](cilium)**: eBPF-based CNI with Gateway API support and L7 proxy capabilities
- **[Cert Manager](cert-manager)**: Cloud-native certificate management for Kubernetes
- **[Vault](vault)**: Secrets management and data protection platform
- **[External Secrets](external-secrets)**: Synchronize secrets from external sources into Kubernetes
- **[ArgoCD](argocd)**: Declarative GitOps continuous delivery for Kubernetes

## Component Details

Each component is thoroughly documented with:

- Version information
- Configuration parameters
- Resource requirements
- Best practices
