Before deploying, make sure your machine meets the basics below. Devbox (or the VS Code
Dev Container) will install all CLI tooling for you.

### System requirements

| Area | Minimum | Comfortable |
| --- | --- | --- |
| CPU | 4 cores | 6+ cores |
| Memory | 8 GB | 12+ GB |
| Storage | 20 GB free | 30+ GB free |
| OS | Linux or WSL2 | MacOS |
