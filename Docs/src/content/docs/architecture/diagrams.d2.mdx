## Architecture Diagrams

This section contains D2 diagrams that visualize the IDP Blueprint architecture.

:::note
D2 diagrams are rendered during the build process. For source files, see the `assets/diagrams/architecture/` directory.
:::

## Available Diagrams

- **Container View**: Shows the major containers (applications and data stores) and their interactions
- **System Context**: High-level view of the system and its external dependencies
- **Component Diagrams**: Detailed views of individual components and their relationships

For more information about the architecture, see the [Architecture Overview](overview).
