## Platform Capabilities

<CardGrid columns={2}>
  <Card title="GitOps Native" icon="lucide:github">
    Everything is code. Infrastructure, applications, and policies are reconciled automatically from Git using **ArgoCD** and **ApplicationSets**.
  </Card>
  <Card title="Policy as Code" icon="lucide:shield-check">
    Guardrails are built-in. **Kyverno** enforces labels, limits, and security contracts at the cluster level before deployment.
  </Card>
  <Card title="Full Observability" icon="lucide:search">
    Day-2 ready. Pre-configured **Prometheus**, **Grafana**, **Loki**, and **Fluent-bit** stacks with dashboards and SLOs managed as code.
  </Card>
  <Card title="Secure by Design" icon="lucide:lock">
    No secrets in Git. **Vault** integrates with **External Secrets Operator** to inject credentials securely into Kubernetes Secrets.
  </Card>
  <Card title="Automated Networking" icon="lucide:network">
    One Gateway to rule them all. **Gateway API** + **Cilium** handle traffic with automated TLS certificates via **cert-manager** and `nip.io`.
  </Card>
  <Card title="Local to Prod" icon="lucide:laptop">
    Bootstraps a full enterprise-like environment on your laptop (via k3d) in **< 10 minutes**, mirroring production constraints perfectly.
  </Card>
</CardGrid>

## Technology Stack

The best-in-class open source components powering the platform.

<CardGrid columns={2}>
  <Card title="Orchestration" icon="lucide:box">
    **Kubernetes** (k3d/k3s) + **Cilium** CNI for eBPF networking.
  </Card>
  <Card title="Delivery" icon="lucide:rocket">
    **ArgoCD** for continuous delivery and **Argo Workflows** for pipelines.
  </Card>
  <Card title="Security" icon="lucide:shield">
    **HashiCorp Vault** for secrets, **Trivy** for scanning, and **cert-manager** for PKI.
  </Card>
  <Card title="Monitoring" icon="lucide:bar-chart-3">
    **Prometheus** & **Grafana** for metrics, **Loki** for logs.
  </Card>
</CardGrid>
