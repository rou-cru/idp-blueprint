# Tutorial: Add an Observability Dashboard

Publish a new Grafana dashboard backed by Prometheus or Loki.

## Steps

1) Identify metrics/logs and build a panel locally
2) Create a dashboard JSON and store it as a ConfigMap/Helm value
3) Add or extend the observability overlay
4) Commit and push to reconcile
5) Validate in Grafana and add to a folder
