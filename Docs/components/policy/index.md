# Policy Components

The policy layer enforces organizational standards and security requirements across the platform.

## Core Components

- **[Kyverno](kyverno/index.md)**: Kubernetes-native policy management and security engine
- **[Policy Reporter](policy-reporter/index.md)**: Monitoring and observability for policy engine results

## Policy Capabilities

These components provide:

- Policy enforcement as code
- Compliance monitoring
- Automated security checks
- Admission control
